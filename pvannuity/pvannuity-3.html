What's the PV of an ordinary annuity that pays $100 per year for 3 years at 10% interest?

The answer is \(PVA=\$248.69\)

Steps to Solve:

  • Identify the Basic Variables

    • First, break down the question. PVA is the value of the annuity in today's dollars. This is the variable we must calculate. n is the measure of time in years. This means n = 3. CF is the value of the regularly scheduled cash flow or payment. This means CF = $100. The % sign is an indicator of interest. This means that i = 10%.

  • Plug the Variables into the Equation

    • Next, plug the numbers into the equation. Use the equation that we have modified to include compounding.

    \(PVA=CF[\frac{1-\frac{1}{(1+i)^n}}{i}]\)
    \(PVA=100[\frac{1-\frac{1}{(1+.10)^3}}{.10}]\)
  • Perform the Algebra

    • Solve the equation using the order of operations. Remember PEMDAS.

    • Start with the numbers in parentheses. Since brackets function like parentheses, work on all the math inside the brackets before moving outside the brackets. \(PVA=100[\frac{1-\frac{1}{(1+.10)^3}}{.10}]\) ⟶ \(PVA=100[\frac{1-\frac{1}{(1.10)^3}}{.10}]\)
    • Raise the number within the parentheses to the exponent. Note: If you are using a calculator, it will be helpful to know that exponents in the denominator can be brought up to the numberator by switching the sign (+/-). Example: \(\frac{1}{(1.10)^3}=(1.10)^{-3}\)
      \(PVA=100[\frac{1-\frac{1}{(1.10)^3}}{.10}]\) ⟶ \(PVA=100[\frac{1-\frac{1}{(1.331}}{.10}]\)
    • Simplify the embedded fraction.
      \(PVA=100[\frac{1-\frac{1}{(1.331}}{.10}]\) ⟶ \(PVA=100[\frac{1-0.75131}{.10}]\)
    • Subtract.
      \(PVA=100[\frac{1-0.75131}{.10}]\) ⟶ \(PVA=100[\frac{0.24860}{.10}]\)
    • As pointed out before, brackets function like parentheses, so simplify the fraction within the brackets next.
      \(PVA=100[\frac{0.24860}{.10}]\) ⟶ \(PVA=100[2.48685]\)
    • Multiply the remaining numbers to find PVA.
      \(PVA=100[2.48685]\) ⟶ \(PVA=248.69\)
