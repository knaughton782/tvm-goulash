Steps to Solve for the Present Value of Annuities:

  • Identify the Basic Variables

    • First, break down the question. PVA is the value of the annuity in today's dollars. This is the variable we must calculate. n is the measure of time in years. This means n = 3. CF is the value of the regularly scheduled cash flow or payment. This means CF = $100. The % sign is an indicator of interest. This means that i = 10%.

  • Plug the Variables into the Equation

    • Next, plug the numbers into the equation. Use the equation that we have modified to include compounding.

    \(PVA=CF[\frac{1-\frac{1}{(1+i)^n}}{i}]\)
  • Perform the Algebra

    • Solve the equation using the order of operations. Remember PEMDAS.

    • Start with the numbers in parentheses. Since brackets function like parentheses, work on all the math inside the brackets before moving outside the brackets.
    • Raise the number within the parentheses to the exponent. Note: If you are using a calculator, it will be helpful to know that exponents in the denominator can be brought up to the numberator by switching the sign (+/-).
      Example: \(\frac{1}{(1.10)^3}=(1.10)^{-3}\)
    • Simplify the embedded fraction.
    • Subtract.
    • As pointed out before, brackets function like parentheses, so simplify the fraction within the brackets next.
    • Multiply the remaining numbers to find PVA.

    Additional Steps for Combination Calculations

    Perform the calculations separately, starting with the annuity. The steps are the same as for all the formulas. Identify the variables, input them into the equation and then work the equation. Then perform the calculations for the lump sum. Add the Present Value of the annuity (PVA) and lump sum (PV) to find the answer.
