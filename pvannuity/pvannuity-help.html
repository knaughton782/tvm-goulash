Annuity (PVA) Calculation

  • P: Since brackets function like parentheses, so we will first simplify the numbers in the parentheses and then work on the bracket as a whole.
  • E: Raise the number within the parentheses to the exponent.

  • MD: Simplify the embedded fraction.
  • AS: Subtract the numbers in the numerator.
  • MD: Simplify the equation within the bracket and multiply the CF by the number within the bracket.

Lump Sum (PV) Calculation

  • P: Start with the numbers in the parentheses. Since brackets function like parentheses, we will work in the following order: inner parentheses, outer brackers, and then outside the bracket.
  • E: Raise the number within the parentheses to the exponent.

  • MD: Divide both sides by the number in the parentheses to isolate PV. Then solve for PV.
  • AD: Add the Present Value of the annuity (PVA) and lump sum (PV) to find what the bond is worth today.
