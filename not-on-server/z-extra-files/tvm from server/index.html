Time Value of Money Tutorials

The following tutorials provide an introduction to Time Value of Money (TVM), developed by Professor Brad Scott.

The tutorials below break TVM into four sections: the basics, compound interest, cash flows, and annuities. Since each section builds on the last, it is recommended that you complete the tutorials in the order presented and move on only when you have a firm grasp of the previous section. The estimated time to complete the entire tutorial is 90 minutes, with additional time to complete the activities at the end of each section.

Learning Outcomes
Upon the completion of the TVM tutorials, students will:

  • Identify variables presented in a TVM word problem
  • Use the TVM equation to calculate Present Value (PV), Future Value (FV), interest rate (i), number of periods (n), and compounding (m)
  • Find the Future Value factor, given an interest rate and number of years Or given the Present Value factor
  • Find the Present Value Factor, given an interest rate and number of years OR given the Future Value factor
  • Draw timelines to represent events described in a TVM word problem
  • Calculate EAR, given a nominal rate with compound interest
  • Discern between an ordinary annuity and an annuity due
  • Solve for the Present Value of Annuities, both with a financial calculator and on your own

Prerequisites
Manipulating the TVM equations requires working knowledge of algebra. In particular, you may need to brush up on the following:

  • FOIL (First, Outer, Inner, Last)
  • Order of Operations (PEMDAS: Parentheses, Exponents, Mulitplication & Division, Addition & Subtraction)
  • Rounding and Significant Digits

Some help has been provided on these topics within the modules.
